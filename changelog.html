Mucho Plugin Changelog

1.3.0 -- September 26, 2009

  • Added setting "plugin.mucho.conferenceService" to be able to specify conference service name if having multiple conference services. If this is not set, the "first" service name in the service name list will be used.

1.2.0 -- May 23, 2009

  • Added setting "plugin.mucho.showSenderJID" that shows JID of the sender of the original message to the bot.

1.1.0 -- Feb 24, 2009

  • Added command "@sayto: <jid> <body>" to be able to make bot say stuff.
  • Added debug flag to properties.

1.0.1 -- Feb 16, 2009

  • New rooms wasn't unlocked - fixed.

1.0.0 -- Feb 16, 2009

  • Initial release.
